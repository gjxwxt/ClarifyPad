# 07. 调研来源链接

以下链接用于本次初步市场扫描。它们说明当前市场中已经存在语音输入、系统级写作工具、跨 App 改写工具等相关产品，因此本项目需要避免只做通用润色。

## 语音输入 / 语音助手

- 闪电说：https://shandianshuo.cn/
- Wispr Flow：https://wisprflow.ai/

## 跨 App AI 写作/改写

- RewriteBar：https://rewritebar.com/
- Selected AI：https://selectedai.app/
- Rewrait：https://apps.microsoft.com/detail/xp9mbxhb03tdl0?gl=CN&hl=zh-CN
- WritingTools 开源项目：https://github.com/theJayTea/WritingTools

## 系统级 / 平台级 AI 写作能力

- Apple Intelligence Writing Tools：https://www.apple.com/cm/newsroom/2024/06/introducing-apple-intelligence-for-iphone-ipad-and-mac/
- Microsoft Notepad AI：https://support.microsoft.com/en-us/windows/enhance-your-writing-with-ai-in-notepad-4088b954-c97b-46dc-813f-959be01746d5

## 写作助手

- Grammarly：https://www.grammarly.com/

## 关键启发

1. 闪电说和 Wispr Flow 证明“粗糙输入自动整理”是强需求，但主要起点是语音。
2. Apple / Microsoft 证明系统级 AI 写作会成为基础能力，通用改写会越来越不稀缺。
3. Grammarly / RewriteBar / Selected / WritingTools 证明跨 App 改写成立，但大多偏“已有文本处理”。
4. 本项目应把差异化放在“提交前思考中间层、缺口提醒、任务化输出、确认记录与个人记忆”上。
