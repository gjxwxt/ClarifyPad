# 04. MVP 路线图与验证计划

## 1. 总体策略

先验证“使用意愿”和“场景刚性”，再投入大量精力做系统级兼容。

不要一开始就做完整输入法。完整输入法涉及系统输入法框架、兼容性、候选栏、权限、隐私、跨平台等复杂问题，MVP 成本过高。

推荐路线：

1. 低保真原型验证需求；
2. Raycast / Alfred / 浏览器插件验证任务模式；
3. macOS 浮层客户端验证真实工作流；
4. Windows 和企业场景后置。

## 2. 阶段 0：用户访谈与样本收集

周期：1-2 周

目标：
- 找到真实高频场景；
- 收集用户原始输入和最终发送文本；
- 验证用户是否有“发送前不够清楚”的痛点。

访谈对象：
- 5 个产品经理；
- 5 个运营/市场；
- 5 个管理者；
- 5 个销售/客服；
- 5 个 AI 高频用户。

关键问题：
- 你最近一次反复修改工作消息是什么场景？
- 什么内容你会复制到 ChatGPT 改？
- 你在发消息前最担心什么？
- 你是否会保存自己写得好的表达？
- 如果一个浮层能在发送前帮你归纳，你愿意多按一次快捷键吗？

输出物：
- 50 条原始草稿；
- 50 条最终表达；
- 20 条典型痛点；
- 10 个高频模式；
- 第一版 prompt 集。

## 3. 阶段 1：Figma / Web Demo

周期：1 周

目标：
- 只验证交互和输出，不做系统能力；
- 让用户感受“中间层”。

功能：
- 输入框；
- 模式选择；
- AI 输出；
- 缺口提醒；
- 确认按钮；
- 历史列表。

验证指标：
- 任务完成率；
- 结果满意度；
- 是否愿意在真实工作中使用；
- 哪些模式最有用。

## 4. 阶段 2：Raycast / Alfred 插件原型

周期：1-2 周

目标：
- 用最小成本进入真实桌面工作流；
- 验证“快捷键 + AI + 粘贴”的链路。

功能：
- 快捷键打开输入框；
- 输入粗糙想法；
- 调用 LLM；
- 复制/粘贴结果；
- 本地保存历史。

优点：
- 开发快；
- 不用自研全局浮层；
- 用户安装门槛较低；
- 可以快速迭代 prompt。

缺点：
- 光标旁弹窗体验弱；
- 品牌感和完整性不足；
- 插入体验依赖工具能力。

## 5. 阶段 3：macOS 原生/Tauri 客户端

周期：3-6 周

目标：
- 实现真正的光标旁输入中间层；
- 验证能否替代复制到 ChatGPT。

技术选型建议：
- Swift / AppKit：系统集成最佳；
- Tauri + Rust：跨平台潜力好，体积小；
- Electron：开发快但重，不建议长期首选；
- 初版可用 Swift + WebView 混合。

功能清单：
- 全局快捷键；
- 浮层定位；
- 文本输入；
- AI 输出；
- 一键插入；
- 历史记录；
- 设置页；
- 隐私黑名单。

## 6. 阶段 4：私测

周期：2 周

人数：30-50 人

选择标准：
- 每天工作 IM / 邮件 / 任务系统使用频繁；
- 愿意反馈；
- 能接受早期产品不稳定。

关键数据：
- 日均唤起次数；
- 插入率；
- 哪些 App 使用最多；
- 哪些模式使用最多；
- 插入失败率；
- 用户主动反馈的场景。

## 7. 阶段 5：小范围公开 Beta

周期：4-8 周

目标：
- 打磨稳定性；
- 明确定价；
- 找到核心人群。

可尝试定价：
- 免费：每天 20 次；
- Pro：¥19-39/月；
- Power：¥59-99/月，支持更多模型、自定义模板、历史增强；
- Team：按席位，团队模板与合规功能。

## 8. MVP 必须回答的 5 个问题

1. 用户是否高频使用？
2. 哪个场景最刚需？
3. 输出是否真的减少沟通成本？
4. 缺口提醒是加分还是干扰？
5. 记忆/历史是否创造长期粘性？

## 9. 建议优先级

### P0

- 全局快捷键
- 浮层输入
- AI 输出
- 一键插入
- 历史保存
- 隐私基础设置

### P1

- 缺口提醒
- 多模式
- 多版本
- 搜索历史
- 自定义快捷键
- 黑名单 App

### P2

- 个人风格学习
- 模板库
- 本地模型
- 选中文本上下文
- 导出记录

### P3

- 屏幕感知
- 团队版
- 自动化技能
- 多端同步
- 移动输入法

## 10. 主要里程碑

| 阶段 | 目标 | 交付物 |
|---|---|---|
| M0 | 需求确认 | 访谈记录、场景样本、竞品分析 |
| M1 | 原型验证 | Figma/Web Demo、Prompt 样例 |
| M2 | 工作流验证 | Raycast/Alfred 插件 |
| M3 | 桌面 MVP | macOS 客户端 |
| M4 | 私测 | 30-50 人使用数据 |
| M5 | Beta | 官网、安装包、付费试验 |
