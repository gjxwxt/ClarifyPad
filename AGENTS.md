# ClarifyPad AGENTS

## Language

- 默认使用中文交流。

## Start Here

- 进入项目先读 `AGENTS.md`。
- 再读 `tasks/index.md` 获取项目记忆入口。
- 开始相关任务前，按需查看 `tasks/lessons.md` 和 `tasks/knowledge.md`。

## Project Defaults

- 当前阶段先做 `v0.0` 技术探针，再做 `v0.1` AI 最小闭环。
- `v0.0` 优先验证系统链路：快捷键、浮层、插入、fallback、埋点。
- 在 `v0.0` 阶段，不要先为 AI 能力、复杂上下文读取、深度个性化扩 scope。
- 当前采用 `Windows-first` 验证顺序，但架构必须从第一天按跨平台设计。
- 平台能力通过适配层隔离，不要把 Windows 或 macOS API 直接写进产品核心逻辑。

## Memory Routing

- 任务状态写入 `tasks/todo.md`。
- 可复用经验、坑点、协作修正写入 `tasks/lessons.md`。
- 长期有效的产品事实、架构决策、稳定约束写入 `tasks/knowledge.md`。
